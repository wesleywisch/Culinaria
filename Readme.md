  <h2 id="sobre"> 📋 Sobre o projeto</h2>
  Esse é um projeto sobre culinária, para você que gosta de cozinhar e não quer perder as suas receitas, esse é um site aonde você possa adicionar a sua receita e abrindo o modal que aparece os ingredientes e a forma de se realizar, e com rapidez com o campo de busca consegue achar aquela receita de forma muito mais pratica e rapida.

<h2 id="aplicacao"> 🎲  Rodando essa aplicação:</h2>

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:  [Git](https://git-scm.com/),  [Node.js](https://nodejs.org/en/),  [Yarn](https://yarnpkg.com/). Além disto é bom ter um editor para trabalhar com o código como  [VSCode](https://code.visualstudio.com/).

```
# Clone este repositório
$ git clone https://github.com/wesleywisch/Culinaria.git

# Instale as dependências
$ yarn ou npm install

# Execute a aplicação
$ yarn start

# Execute o backend
$ yarn server

# O app estará disponível no seu browser pelo endereço - http://localhost:3000
```
